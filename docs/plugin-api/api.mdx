# API
Provides useful methods for mdpkm's functionality,
most of this class will not be documented.

Accessible as a global variable in the plugin environment.

## Methods
### get
**get**(`id: string`): [APIPlatform](classes/api-platform) | [Loader](classes/loader)

Returns a registered API.

If you're getting a loader use [getLoader](#getLoader) instead.

#### Examples
```js title="Getting a project from Modrinth"
const data = await API.get('modrinth').getProject('fabric-api');
console.log(data);
```

### getLoader
**getLoader**(`id: string`): [Loader](classes/loaderm)

Returns a registered loader.

#### Examples
```js title="Getting a registered loader"
// Logs Quilt's recommended mod to the console
const quilt = await API.getLoader('quilt');
console.log(quilt.recommendedMod);
```

### makeRequest
**makeRequest**(`url: string`, `options:` [RequestOptions](interfaces/request-options)?): <a href={PROMISE_LINK}>Promise</a><[WebResponse](interfaces/webresponse)>

Makes a request to the specified URL with the provided options.

#### Examples
```js title="Sending a GET request"
const data = await API.makeRequest('https://httpbin.org/get');
console.log(data);
```
```js title="Sending a POST request"
const data = await API.makeRequest('https://httpbin.org/get', {
    // https://tauri.studio/v1/api/js/classes/http.Body
    // Yes, http is defined in the plugin environment globally!
    body: http.Body.json({
        message: 'hello!'
    }),
    method: 'POST'
});
console.log(data);
```
